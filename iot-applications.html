Activity 1

What is an IoT application?

    An IoT application or Internet of Things are applications which are 
    used to control the hardware through the use of a simple interface 
    build using web technologies such as python, html, css, js etc.

What are the main Features needed?

    
      Data Security 
      Realtime data monitoring
      Scalability
      Integration
      Realtime data monitoring
      Scalability
      Integration

      Smart evacuation system
      Health monitoring system

Activity 2

What does smart mean?

    
      Smart building refers to a building which is automated to a certain extent 
      in order to reduce human effort and control appliances using an interface 
      while also covering faults that may occur during the whole process.

What are the main Features needed?

    
      Data handling: data security, data backup, data access restriction
      Sensor data precision and accuracy
      Realtime health monitoring
      Fault notifying system

What are some Smart-building scenarios?

    
      Security (Intruder detection)
      Congestion avoidance detection
      Air quality monitoring and ventilation
      Turning on and off appliances automatically based on its usage hence saving electricity.
